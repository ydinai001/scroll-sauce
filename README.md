# Scroll Sauce

Short-form video editing service for creators who want scroll-stopping content.

**Live Preview:** https://adorable-zabaione-493e62.netlify.app/

## Service Overview

- **Price:** $40 per reel
- **Turnaround:** 3 days
- **Platforms:** Instagram Reels, TikTok

## Project Structure

```
scroll-sauce/
├── index.html          # Landing page mockup
├── assets/
│   ├── images/         # Photography and graphics
│   └── icons/          # Icon assets
├── css/                # Stylesheets (future)
└── js/                 # Scripts (future)
```

## Tech Stack

- HTML5 / CSS3
- Google Fonts (Syne, Space Mono)
- Unsplash imagery
